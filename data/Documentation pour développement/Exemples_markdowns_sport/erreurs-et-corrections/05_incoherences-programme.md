# Incohérences dans un programme d'entraînement

## Erreurs fréquentes

- 3 séances push / 1 tirage
- Aucun jour jambes
- Volume disproportionné entre muscles

## Solutions

- Répartition logique (push/pull, haut/bas)
- Analyser le volume / fréquence / intensité
- Intégrer les jambes et muscles posturaux

## Cas d’usage typiques

- Détection automatique d’erreurs dans un générateur IA
- Audit de programme manuel ou semi-automatisé
- Guide de structuration pour coach débutant
