# Programme Push Pull Legs

## Structure

- Push : Pecs, épaules, triceps  
- Pull : Dos, biceps  
- Legs : Jambes complètes

## Exemple 3 jours

| Jour      | Séance |
|-----------|--------|
| Lundi     | Push   |
| Mercredi  | Pull   |
| Vendredi  | Legs   |

## Exemple 6 jours

| Jour      | Séance |
|-----------|--------|
| Lundi     | Push   |
| Mardi     | Pull   |
| Mercredi  | Legs   |
| Jeudi     | Push   |
| Vendredi  | Pull   |
| Samedi    | Legs   |

## Cas d’usage typiques

- Utilisateurs avec volume d'entraînement élevé
- Intégration de surcharge progressive sur chaque chaîne musculaire
- Planning structuré pour IA générative ou recommandation automatique
- Adaptation selon objectif (volume ou force)
- Transition vers du bro-split personnalisé
