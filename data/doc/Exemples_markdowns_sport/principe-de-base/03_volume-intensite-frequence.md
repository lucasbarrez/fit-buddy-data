# Volume, Intensité, Fréquence

## Définitions

- **Volume** : quantité totale de travail (séries x répétitions x charge)
- **Intensité** : niveau d’effort (souvent exprimé en % de 1RM)
- **Fréquence** : nombre de fois qu’un groupe musculaire est travaillé par semaine

...

## Cas d’usage typiques

- Créer des programmes spécifiques à un objectif (force, hypertrophie, endurance)
- Ajuster un plan existant pour améliorer la progression
- Intégrer une logique intelligente dans un assistant de coaching
- Éviter le surentraînement via la gestion du volume total
- Optimiser un plan hebdomadaire selon disponibilité de l’élève
