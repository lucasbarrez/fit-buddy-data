# Pré-fatigue & Post-fatigue

## Définition

- **Pré-fatigue** : isolement → polyarticulaire  
- **Post-fatigue** : polyarticulaire → isolement

## Objectif

Recruter plus intensément un muscle cible dans un mouvement complexe.

## Exemples

- **Pré-fatigue** : écarté poulie → développé couché  
- **Post-fatigue** : squat → leg extension

## Avantages

- Meilleure congestion
- Travail ciblé malgré la fatigue globale
- Permet de "sentir" le muscle

## Cas d’usage typiques

- Correction d’un pattern moteur
- Isolement d’un muscle faible (ex : pectoraux qui ne recrutent pas)
- Variation intelligente dans programme IA
- Fin de cycle hypertrophie
