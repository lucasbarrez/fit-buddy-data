# Bro Split

## Structure type

| Jour      | Groupe |
|-----------|--------|
| Lundi     | Pecs   |
| Mardi     | Dos    |
| Mercredi  | Jambes |
| Jeudi     | Épaules|
| Vendredi  | Bras   |

## Avantages

- Concentration maximale sur un muscle
- Volume d'entraînement important
- Congestion et fatigue localisée élevée

## Limites

- Faible fréquence de stimulation musculaire
- Recommandé uniquement si récupération parfaite

## Cas d’usage typiques

- Bodybuilder expérimenté
- Plan personnalisé selon points faibles
- Application de techniques avancées (dropsets, pré-fatigue)
- Analyse IA des préférences d’utilisateur avancé
- Génération de split via assistant virtuel
